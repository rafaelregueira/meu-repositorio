# Projeto Pessoal - Página Web Pessoal

Este projeto é uma página HTML simples que apresenta um perfil pessoal com nome, foto e redes sociais.

## Tecnologias Usadas
- HTML5
- CSS3

## Como Usar
1. Clone este repositório.
2. Abra o arquivo `index.html` no navegador.
